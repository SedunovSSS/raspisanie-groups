Получение расписания с помощью API

В URL адресе есть 2 аргумента group - Группа и date - Дата

Если перейти на такую страницу то можно увидеть:

Если расписание не найдено:

Для получения расписания с помощью API будет использоваться язык Python

Для начала нужно установить библиотеку

requests

 командой pip install requests


Дальше создадим файл api_example.py и откроем его в любом текстовом редакторе

В нём необходимо написать код для получения API


Переменная date отвечает за дату в формате '1999-01-01' или если нужно расписание на сегодня 'today'

Переменная group отвечает за группу

Если расписание существует, то скрипт выведет его


Если расписание не существует, то скрипт выведет ошибку


С помощью библиотеки Tkinter можно добавить графический интерфейс
